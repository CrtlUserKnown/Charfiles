## Release v0.4.0

This release includes several updates to Neovim, tmux, and Zsh configurations.

### Features

-   **Neovim:**
    -   Switched to `Lazy.vim` package manager from `packer` for better plugin support.
    -   Various other changes to Neovim configuration.
-   **Zsh:**
    -   Fixed Zsh completions.
    -   Removed `zim` from the configuration.
-   **Other:**
    -   Added configurations for `bat`, macOS terminal, and `fzf`.
    -   Updated `ghostty` theme for Tahoe.
    -   Corrected symlinks in `install.sh`.

### Fixes

-   Updated `README.md`.
